# M018: 

## Vision
Make ThreadMoat a reference tool (not just exploration), gate data for a permanent free tier that fuels the go-to-market flywheel, smooth the upgrade/downgrade paths between plans, and add a scenario-aware narrative thread so users know what to look at and why.

## Slice Overview
| ID | Slice | Risk | Depends | Done | After this |
|----|-------|------|---------|------|------------|
| S01 | Company Search & Profile | medium | — | ⬜ | User types a company name in the search bar, sees fuzzy results, clicks one, and views a curated profile with expandable sections showing all ThreadMoat data for that company. |
| S02 | Free Preview Tier | high | S01 | ⬜ | A free Recon user logs in, sees 50 companies in search and charts, sees 'Showing 50 of 500+ companies' indicators, and sees upgrade CTAs on gated content. No 30-day expiry. |
| S03 | Plan Transition Polish | medium | S02 | ⬜ | A Strategist subscription expires. If the user previously purchased Analyst, they fall back to Analyst access. If not, they fall back to Recon. Expiry banner shows 14 days before renewal. Pricing page reflects permanent free tier. |
| S04 | Scenario-Aware Narrative Thread | medium | S01 | ⬜ | User selects 'White Space Filler' scenario. Dashboard landing shows an intro narrative explaining the analytical journey. Each chart page has a scenario-specific annotation explaining what to look for. A 'Next: See funding patterns →' link guides to the logical next chart. |

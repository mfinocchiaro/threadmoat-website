    **1. Create `lib/metadata.ts`** with two exported functions:

    `buildAlternates(locale: string, pagePath: string)` — returns `{ canonical: string, languages: Record<string, string> }`. For each locale in `routing.locales`, build URL as:
    - Default locale (en): `https://threadmoat.com{pagePath}` (no `/en/` prefix)
    - Other locales: `https://threadmoat.com/{locale}{pagePath}`
    - For home page (`/`), path portion is empty string (not trailing slash)
    - Add `x-default` pointing to same URL as default locale
    - Canonical URL follows same logic based on current locale

    `buildOpenGraph(title: string, description: string, locale: string, pagePath: string)` — returns `Metadata['openGraph']` with fields: `title`, `description`, `url` (computed like canonical), `siteName: 'ThreadMoat'`, `locale`, `type: 'website'`.

    Import `routing` from `@/i18n/routing`. Use `const BASE_URL = 'https://threadmoat.com'`.

    **2. Create `app/sitemap.ts`** — export default function returning `MetadataRoute.Sitemap`. Loop over `PUBLIC_PAGES = ['/', '/pricing', '/about', '/report']` and for each page:
    - Build `alternates.languages` for all 6 locales (same URL logic as buildAlternates)
    - Add `x-default`
    - Set `url` to the default locale URL
    - Set `lastModified: new Date()`, `changeFrequency: 'weekly'`, `priority: 1.0` for home, `0.8` for others

    **3. Create `app/robots.ts`** — export default function returning `MetadataRoute.Robots`:
    ```
    rules: { userAgent: '*', allow: '/', disallow: ['/api/', '/auth/', '/dashboard/'] }
    sitemap: 'https://threadmoat.com/sitemap.xml'
    ```

    **4. Delete `public/robots.txt`** — the static file takes precedence over `app/robots.ts`. Remove it so the dynamic version is served.

    **5. Update `app/layout.tsx`** — add `metadataBase: new URL('https://threadmoat.com')` to the existing `metadata` export. Keep all existing fields (title, description, icons, generator). Just add `metadataBase` as the first field.

## Milestone Summary

**Key Decisions:**
- next-intl for i18n with localePrefix: as-needed (no /en redirect)
- Tiers renamed: Forge→Analyst, Red Keep→Strategist, added Advisory
- Analyst is one-time $4,999 purchase, Strategist is €18,999/yr subscription
- intlMiddleware runs outside NextAuth wrapper to prevent rewrite→redirect conversion
- React Email + Resend for transactional emails
- Admin purchase notification for watermarked report delivery

**Issues Resolved:**
- Production /en redirect 404 (intlMiddleware inside auth() wrapper)
- Auth link locale-prefix 404s (used NextLink for non-locale routes)
- OG image locale cookie redirect (cookie stripped for crawler requests)
- Stale sandbox Stripe customer IDs (validated before use)
- robots.txt 404 (bypassed proxy middleware)

**Technical Debt:**
- Filter bar needs redesign from dialog overlay to sticky toolbar
- French/Portuguese translations pending native speaker review
- Stripe upgrade coupon needed for report→subscription upgrades
